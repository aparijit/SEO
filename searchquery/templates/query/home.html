How it works

Enter a search query relevant to the topic of your webpage. Our tool grabs the results from page one of Google (top 10 results), extracts out the relevant keywords from each page, and then shows dependencies among those keywords using a network. Results from this network can be used to boost your webpage's ranking.

Under the hood

We use Python programming language to implement all the underlying codes and Python based Django to power up this site!

  • We have used the our Google Results Extractor module to extract the relevant text from the HTML of each page. This algorithm can be used to detect and remove the surplus "clutter" (boilerplate, templates) around the main textual content of a web page.
  • We have used the Keyword Extractor module to identify the relevant keywords from text in English. Keywords can be compounded by one or more words and are defined as the important topics in the content.

Further Improvements

This app is under development process. We have experimented the algorithm on Meta Descriptions. The same can be extended for title and description and other parameters. Several other extensions such as automatic sentence generation from the graph can be done.
